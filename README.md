# DCS localhost utility

This project allows you to export real-time telemetry from **DCS World** to an external application using **Lua** and **LuaSocket** via **UDP**, Similar to the localhost:8111 seen in **War Thunder**, which allows you to read flight information.


---

## Features

- Exports player flight information:
  - Aircraft and pilot name
  - Speeds (IAS/TAS)
  - Altitudes (MSL/AGL)
  - Flight angles (pitch, roll, yaw)
  - Vertical speed
  - Mach number and angle of attack
  - Geographic coordinates (latitude/longitude)
- Real-time, low FPS impact
- Compatible with any DCS aircraft
